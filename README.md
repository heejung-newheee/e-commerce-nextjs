# 회원가입/로그인 Next.js 프로젝트

Next.js(with TypeScript)프로젝트로 라이브러리들을 사용해 간단한 회원가입 로그인 페이지를 구현한 프로젝트입니다.
shadcn/ui을 이용해 ui를 구현하였고 react hook form과 zod를 이용해 유효성 검사를 구현하였습니다.

1. 회원가입/로그인(진행중)

3. 라이브러리를 이용한 유효성 검사
4. 애니메이션 전환 효과
